== Description ==

Let's you fully customize tweets that readers share using your Tweet button. Add custom #tags, @tags, shortened URLs, recommend Twitter accounts to follow and more.


[Support](http://www.dragonsearchmarketing.com/contact-dragonsearch-marketing/)

Features:


* Customize the full body of the default tweet for each post, including the addition of hashtags, @ tags, and more.

* Customize preferred “via” Twitter handle(s)

* Recommend Twitter accounts to follow that the user will see in the window after posting a tweet.

* Use your own custom URL if desired.

* Choose location and size of Twitter button


Follow us on Twitter to keep up with the latest updates - [@DragonSearch](http://twitter.com/DragonSearch/)

== Installation ==

You can use the built in installer or you can install the plugin manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your Setting Options (*Settings - Advanced Twitter*)
4. Configure as desired.
5. To place the button anywhere in your template use - <?php echo get_advanced_tweet(); ?>
4. That's it!
